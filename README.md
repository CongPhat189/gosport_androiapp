# GoSport Android App

GoSport is an Android application developed to support sports field booking and management. The application allows users to browse sports facilities, book fields, make payments, and manage their booking history through a user-friendly mobile interface.

## Features

### User Features

* Register account with input validation
* Login and authentication
* Browse sports fields(link GG maps)
* View field details by sports category
* Book fields by date
* Fixed schedule booking
* Payment support:

  * Cash
  * Bank transfer(VietQR)
  * E-wallet(VNPay)
* Review and rate sports fields
* Delete user account

### Admin Features

* User management

  * Add, edit, delete users
  * Role management
* Sports category management
* Sports field management
* Booking/order management
* Revenue statistics:

  * Daily
  * Monthly
  * Quarterly
  * Yearly

### Additional Features

* Friendly and easy-to-use interface
* Local database management with SQLite
* Responsive Android UI design

## System Architecture

The application is developed following Android application development practices with:

* Activity & Fragment structure
* SQLite local database
* MVVM Architecture
* Custom adapters and reusable components

## Installation

1. Clone the repository:

```bash
git clone https://github.com/CongPhat189/gosport_androiapp.git
```

2. Open project in Android Studio

3. Sync Gradle dependencies

4. Run the application on emulator or Android device
